## 2Ô∏è‚É£ Optional Positional Parameters

## 3Ô∏è‚É£ Named Parameters (Optional)
```dart
void volumeNamed({double? length, double? breadth, double? height}) {
  print((length ?? 1) * (breadth ?? 1) * (height ?? 1));
}

void main() {
  volumeNamed(length: 5, breadth: 3, height: 2);
}
```

## 4Ô∏è‚É£ Required Named Parameters
```dart
void volumeRequiredNamed({

  required double breadth,
  required double height,

  print(length * breadth * height);
}

void main() {
  volumeRequiredNamed(length: 5, breadth: 3, height: 2);
}
```

## 5Ô∏è‚É£ Default Parameters

### üß† One-line Rule to Remember

* `()` ‚Üí required positional
* `[]` ‚Üí optional positional
* `{}` ‚Üí named
* `required` ‚Üí compulsory named
* `=` ‚Üí default value
